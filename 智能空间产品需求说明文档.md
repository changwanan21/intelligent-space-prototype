## 1. 产品概述

### 1.1 产品定位

**智能空间**是一款面向企业及专业用户的 AI 驱动数据分析平台。用户可通过"自定义场景"的方式，将数据集市中的数据集与模型集市中的 AI 模型进行组合，创建专属的数据分析"空间"，在空间内完成数据浏览、检索、AI 字段扩展与可视化分析大屏搭建。

### 1.2 核心价值

| 价值维度 | 说明 |
|---------|------|
| **降低门槛** | 非技术用户无需编写代码，通过图形化配置即可完成数据与模型的联结 |
| **数据增值** | AI 模型对原始数据进行字段扩展（如技术分类、创新评分），使原始数据具备更高分析价值 |
| **洞察加速** | 内置可视化分析与 AI 报告生成，缩短从数据到洞察的路径 |
| **灵活复用** | 数据集市与模型集市解耦，支持多数据集与多模型的自由组合 |

### 1.3 目标用户

- 企业专利/知识产权部门分析人员
- 产业研究机构、高校科研人员
- 企业战略与市场情报团队
- 对特定领域数据有深度分析需求的专业人士

## 2. 用户角色与使用场景

### 2.1 用户角色

| 角色 | 描述 | 典型诉求 |
|------|------|---------|
| **普通用户（专业版）** | 独立订阅用户，拥有个人工作空间 | 创建、管理自己的空间；检索数据；查看 AI 分析结果 |
| **管理员** | 企业账号管理员 | 管理数据集授权；统一分配 AI 模型配额 |

### 2.2 典型使用场景

**场景 A：专利竞争分析**  
某企业知识产权分析师，需要定期对竞争对手的专利布局进行监控与分析。他从数据集市选取「中国专利数据库」，结合技术分类模型，创建「竞品专利追踪」空间，通过 AI 字段识别竞对的技术方向，并将分析大屏固定展示。

**场景 B：行业技术趋势研究**  
某科研机构研究员，需要分析特定 IPC 分类下近三年的专利趋势。他上传自定义专利数据集（CSV 格式），配合创新评估模型，在空间中生成技术趋势折线图并导出分析报告。

**场景 C：快速数据探索**  
新加入团队的分析人员，希望快速了解现有数据集的字段结构和数据分布，通过「数据字段」页签一览基础字段与 AI 字段定义，快速上手数据检索与分析。

## 4. 功能模块详述

### 4.1 首页概览

**功能描述**：用户登录后的默认落地页，提供全局数据汇总与快捷操作入口。

**核心内容**：

#### 4.1.1 统计卡片（4 组）

| 指标 | 说明 |
|------|------|
| 我的空间 | 当前用户创建的空间总数，显示环比增长率 |
| 数据集 | 平台可用数据集总量，显示近期新增数 |
| AI 模型 | 平台可用模型总量，显示近期上线数 |
| 分析任务 | 当前活跃的分析任务数量及状态 |

#### 4.1.2 快捷操作（3 项）

| 操作 | 说明 |
|------|------|
| 创建新空间 | 跳转至自定义创建空间流程 |
| 上传数据集 | 支持 CSV / Excel / JSON 格式的数据上传 |
| 智能分析 | AI 驱动分析的快捷入口 |

#### 4.1.3 我的空间（卡片列表）

在首页以卡片形式展示最近 3 个空间，显示空间名称、描述、数据量、使用模型、运行状态，点击进入对应空间详情页。

---

### 4.2 我的空间

**功能描述**：展示用户所有已创建空间的完整列表，支持状态筛选与管理操作。

**功能点**：

1. **空间列表**：以列表卡片形式展示，每条显示空间名称、描述、数据量、模型、创建时间、状态标签
2. **状态筛选**：支持按"全部 / 运行中 / 已暂停"筛选
3. **编辑**：点击编辑图标，可修改空间名称与描述
4. **删除**：点击删除图标，二次确认后删除空间（不删除关联数据集）
5. **创建入口**：页面右上角提供「创建空间」按钮

**空间状态定义**：

| 状态 | 颜色标签 | 说明 |
|------|---------|------|
| 运行中 | 绿色 | 模型已完成数据计算，空间正常可用 |
| 处理中 | 蓝色 | 模型正在计算 AI 字段 |
| 已暂停 | 灰色 | 暂停模型调用，数据仍可检索 |
| 创建失败 | 红色 | 数据集加载或模型调用失败 |

---

### 4.3 创建空间（自定义场景）

**功能描述**：三步向导式流程，引导用户完成空间的配置与创建。

#### Step 1：填写基本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 空间名称 | 文本输入 | 是 | 不超过 30 字 |
| 空间描述 | 文本域 | 否 | 不超过 100 字 |
| 空间图标 | 颜色/图标选择器 | 否 | 系统提供预置图标与颜色，默认随机分配 |

#### Step 2：选择数据集

两种数据来源，单选：

**来源 A：数据集市**
- 展示平台数据集卡片列表，包含数据集名称、数据量、分类标签、简介
- 支持关键词搜索与分类筛选
- 单次选择 1 个数据集（MVP 阶段）

**来源 B：自定义上传**
- 支持格式：CSV、Excel（.xlsx/.xls）、JSON
- 单文件大小限制：100MB
- 上传后系统自动解析字段结构，用户可预览字段列表
- 解析失败需给出明确的错误提示（如编码问题、格式不支持等）

#### Step 3：选择 AI 模型

- 展示模型集市中可用的模型卡片，包含模型名称、适用场景、预计处理时间、价格说明
- 每个模型需标注**适用数据类型**（如：专利数据、金融数据、通用文本等）
- 支持"不选模型"（仅用数据集的原始字段，不做 AI 扩展）
- 选择后展示：该模型将生成的 AI 扩展字段列表（预览）

#### 创建完成

- 点击"创建"后触发异步任务，系统提示预计完成时间
- 创建成功后跳转至该空间的「数据字段」标签页
- 若创建失败，提供具体失败原因及重试入口

### 4.4 空间详情 - 分析大屏

**功能描述**：空间详情页的默认标签，以数据可视化大屏形式展示当前数据集的核心分析结论。

**设计特点**：深色科技风格（深蓝/深灰渐变背景），毛玻璃卡片效果。

#### 顶部统计卡片（4 组）

针对专利场景，展示：

| 指标卡 | 颜色 | 说明 |
|--------|------|------|
| 专利总数 | 紫色 | 数据集中专利记录总数 |
| 申请主体 | 蓝色 | 不重复的申请人/专利权人数量 |
| 有效专利 | 绿色 | 法律状态为"有效"的专利数 |
| 本年新增 | 橙色 | 当年新申请专利数量 |

> **通用化说明**：统计指标应根据数据集类型动态配置，专利数据集为以上示例，其他类型数据集由后台配置对应的统计口径。

#### 图表区（3 列网格布局）

| 组件 | 位置 | 图表类型 | 数据内容 |
|------|------|---------|---------|
| 技术领域分布 | 左列 | 甜甜圈图（Doughnut） | 按技术分类字段统计占比，配合图例说明 |
| 专利申请趋势 | 中间 2 列 | 折线图（Line） | 近 12 个月申请量与授权量对比，双数据集 |
| 热门申请人 TOP10 | 右列 | 横向排名条形 | 申请量前 10 名，带进度条可视化 |
| 最新专利动态 | 底部 | 卡片列表 | 最新入库的专利摘要（申请号、名称、类型、申请人） |

#### 大屏编辑入口

- 顶部提供"编辑大屏"按钮，跳转至智能分析页进行图表组合
- 大屏内容来源于用户在智能分析页"保存到大屏"的图表

### 4.5 空间详情 - 数据字段

**功能描述**：展示当前数据集的完整字段结构，区分原始字段与 AI 扩展字段，便于用户了解数据口径。

#### 数据集概览卡片（4 栏）

| 信息项 | 说明 |
|--------|------|
| 数据集名称 | 当前关联的数据集名称 |
| 数据总量 | 当前数据集记录总条数 |
| 分析模型 | 当前使用的 AI 模型名称 |
| 更新时间 | 数据集最近一次刷新时间 |

顶部标注"已连接"状态标签，提供"刷新数据"操作按钮。

#### 基础字段（原始数据）

以 3 列网格布局展示，每个字段卡包含：
- 字段名（中文）
- 数据类型（string / date / int / float / enum / array）
- 字段说明

**以专利数据集为例（9 个基础字段）**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 申请号 | string | 专利申请时的唯一标识号 |
| 公开号 | string | 专利公开时的编号 |
| 公开日 | date | 专利公开的日期 |
| 申请人 | string | 提交专利申请的主体 |
| 专利权人 | string | 拥有专利权的主体 |
| 发明名称 | string | 专利的发明名称 |
| 专利类型 | enum | 发明 / 实用新型 / 外观设计 |
| IPC 分类 | string | 国际专利分类代码 |
| 主申请人地址 | string | 申请人所在地区 |

#### AI 分析字段（模型生成）

背景以紫色渐变区分，标注"模型生成"标签，展示 AI 扩展字段：

**以专利智能分析场景为例（6 个 AI 字段）**：

| 字段名 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| 技术分类 | string | AI 自动识别的技术领域分类 | 新一代信息技术、高端装备 |
| 技术关键词 | array | AI 提取的核心技术关键词 | ["5G", "天线", "阵列"] |
| 创新性评分 | float | AI 评估的创新程度（0-100） | 78.0 |
| 技术成熟度 | enum | AI 评估的技术发展阶段 | 探索期 / 成长期 / 成熟应用 |
| 应用领域 | array | AI 识别的潜在应用场景 | ["通信", "物联网"] |
| 法律状态 | enum | AI 分析的法律有效性 | 有效 / 审中 / 无效 |

> **扩展说明**：AI 字段的具体内容由所选模型决定，字段列表在创建空间 Step 3 选择模型时即可预览。不同场景（金融、供应链、用户画像等）会有不同的 AI 字段定义。

### 4.6 空间详情 - 数据检索

**功能描述**：在当前空间数据集范围内，提供灵活的数据检索与浏览能力。

#### 快速检索区

- **关键词搜索**：支持输入申请号、公开号、发明名称等关键词进行全文检索
- **类型筛选**：下拉选择专利类型（全部 / 发明专利 / 实用新型 / 外观设计）
- **状态筛选**：下拉选择法律状态（全部 / 有效 / 审中 / 无效）
- **搜索按钮**：触发检索请求

#### 快捷筛选标签

系统根据数据集特征预置常用筛选条件，展示为标签按钮形式，点击快速过滤：

- 近一年新增
- 高创新评分
- 重点企业
- 高校科研
- 按技术分类快捷筛选（新一代信息技术 / 生物医药 / 等）

#### 搜索结果表格

| 列名 | 类型 | 说明 |
|------|------|------|
| 申请号 | 链接 | 点击可跳转至详情 |
| 发明名称 | 文本（截断显示） | 超长部分 ... 展示 |
| 申请人 | 文本 | — |
| 类型 | 标签 | 颜色区分发明/实用/外观 |
| 技术分类 | 标签（AI 字段） | 颜色区分技术领域 |
| 创新评分 | 数值（AI 字段） | 紫色高亮 |
| 操作 | 按钮 | "查看"跳转详情页 |

- **结果导出**：支持将当前检索结果导出为 Excel / CSV
- **分页**：每页默认显示 20 条，支持跳页

### 4.7 空间详情 - 智能分析

**功能描述**：提供图表自由组合、AI 分析报告生成、图表保存到大屏等能力。

#### 图表组合分析（主区域）

**① 图表类型选择**（4 种）：

| 图表类型 | 适用场景 |
|---------|---------|
| 柱状图 | 分类对比（如按技术领域统计数量） |
| 折线图 | 趋势变化（如按时间统计申请量） |
| 饼图 | 占比分布（如技术分类比例） |
| 散点图 | 相关性分析（如创新评分与申请量关系） |

**② 维度配置**：

| 配置项 | 可选值 |
|--------|--------|
| X 轴维度 | 申请年份 / 公开年份 / 申请人 / 技术分类 / 专利类型 / 地区 |
| Y 轴维度 | 专利数量 / 创新评分均值 / 申请人数量 |

**③ 图表预览区**：实时渲染所选图表类型与数据维度的预览图，高度 220px 固定容器。

**④ 保存到大屏**：点击"保存到大屏"按钮，将当前配置的图表添加到「分析大屏」标签页展示，同时在「已保存图表」列表中增加记录。

#### AI 智能分析报告（辅区域）

- **整体分析**：自动生成数据集规模、技术分布概述
- **趋势洞察**：近期趋势变化、峰值说明
- **创新亮点**：高分段专利特征分析
- **关注建议**：不足领域、转化率提示、法律状态风险提示

报告以结构化卡片展示，支持后续版本扩展为可下载 PDF。

#### 生成新的分析（右侧边栏）

| 配置项 | 选项 |
|--------|------|
| 分析类型 | 技术趋势分析 / 竞争分析 / 地域分析 / 专项报告 |
| AI 模型 | GPT-4 / Claude-3 / GLM-4 等 |
| 分析要求 | 自由文本输入（可选附加条件） |

点击"开始分析"触发后台异步分析任务，完成后刷新分析报告内容。

#### 已保存图表列表

- 以列表形式显示历史保存到大屏的图表（图表类型 + 名称 + 状态）
- 支持删除已保存图表（同步从大屏移除）

### 4.10 模型集市

- 支持"试用"功能（接入 Demo 数据预览模型输出效果）
- 标注模型成熟度（测试中 / 正式上线）

## 5. 数据字段规范

### 5.1 字段数据类型定义

| 类型标识 | 含义 | 示例 |
|---------|------|------|
| `string` | 字符串 | "华为技术有限公司" |
