## 技术细节

- 前端：纯 HTML + Tailwind CSS（CDN）+ Chart.js
- 图表容器必须用 `position:relative` + 固定高度的 div 包裹 canvas，否则比例异常
- 切换标签页时需先销毁旧 Chart 实例再重建，避免叠加
- 图表类型切换使用销毁+重建（不用 config.type 热切换）
